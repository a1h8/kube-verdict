KubeVerdict

Evidence-first Kubernetes incident decision engine.

KubeVerdict turns Alertmanager alerts into evidence-grounded Kubernetes RCA by correlating live Kubernetes events and Helm drift, ranking confidence, estimating remediation impact, and keeping a human approval gate before any command touches production.

Workflow

Alertmanager → Kubernetes evidence → RCA → blast radius → rollback → human approval → recovery.
